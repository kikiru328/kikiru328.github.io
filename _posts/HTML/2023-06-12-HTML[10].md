# Form

```html
<form></form> : 사용자에게 정보를 추가하는 레이아웃
<input ~/> : 사용자에게 정보를 받는 곳.  
```

Input Attributes
> placeholder : 사용자에게 입력을 유도하는 문구
> submit : 제출 버튼
    > value : Change Submit button text
> required : 필수 입력
