# HTML 작성 요령
HTML 문서의 작성 형태가 존재한다.
모든 HTML 문서는

```html
<!DOCTYPE html>
```

로 시작한다.  
이는 브라우저에게 TEXT파일이 아닌 HTML파일 임을 알려주는 역할을 한다.  
그 다음은 아래와 같다.

```html
<!DOCTYPE html>
<html>
    <head>
        ...
        여기에는 브라우저, 웹 사이트의 환경 설정에 대한 내용으로 구성된다. 
        (ex) title
        <title>Home - My first website.</title>
    </head>
    <body>
        ...
        여기에는 사용자가 볼 수 있는 내용으로 구성된다.
    </body>
</html>
```

`브라우저에서 보여질 모든 내용들은 BODY안에 있어야 한다.`
