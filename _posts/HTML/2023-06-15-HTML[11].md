# Form

1. label 은 input과 같이 사용해야 된다.
2. label의 for attribute와  input id attribute는 동일해야 된다.
3. id 는 unique identity. 따라서 한 tag에는 하나의 id만 존재해야 된다.
   
```html
<form>
    <label for="profile">Name</label>
    <input id="profile">'
```
