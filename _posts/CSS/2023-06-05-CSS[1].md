# CSS는 HTML과 같이 써야 한다는 것.  
CSS는 브라우저에게 웹사이트가 `어떻게` 보여지는지 ? 를 의미한다.

* HTML은 browser의 Content를 의미하고  
* CSS는 Content가 어떻게 보여지는지  

# ADD CSS into HTML
1. inline code
2. sperate file

```html
<!-- inline code -->
<head>
    <style>1. css 코드 </style>
    <link href='css 위치' rel='stylesheet'>
```
